# Dennis Snellenberg Portfolio Clone

This is a high-fidelity reconstruction of the portfolio website [dennissnellenberg.com](https://dennissnellenberg.com/).

## Features (To be implemented)
- **Fluid Motion**: GSAP + ScrollTrigger + Locomotive Scroll.
- **Micro-Interactions**: Magnetic buttons, custom cursor, image hover previews.
- **Seamless Navigation**: Barba.js for page transitions.
- **Responsive Design**: Premium typography and layout parity.

## Project Structure
- `/src`: Modular logic and styles.
- `/assets`: Visual resources (fonts, images).
- `/content`: Decoupled project and site data.

## Development
```bash
npm install
npm run dev
```
